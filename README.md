# lowcode
流程配置化-->loader-->流程-->流程节点  
一个状态机就对应包含一个-->流程-->流程节点  

最小交互原则  
减少类型间的交互，减少类型之间的耦合  
减少继承多用组合  
抽离实现通用的拖拽组件，为内部组件提供拖拽能力  
左侧功能列表，自带功能组件Image、text等等  
抽离Node类，每拖拽一个就生成一个Node。各种数据  
实现Editor模型，root为根的递归模型  
中间画布Panel(目前准备使用iframe)，所有用户拖拽之后渲染的结果   
命令行cli工具，用于创建的代码模板
